## Compiling

1. Clone the code
    1. `git clone https://github.com/Milo-C-6/comp3-project.git`
2. Install w64devkit from [/skeeto/w64devkit/](https://github.com/skeeto/w64devkit/releases/latest) and make sure its on path
3. Setup cmake configuration
    1. `cmake -S . -B build`
4. Build
    1. `cmake --build build`

Executable file will be in  (`../build/comp3-project/comp3-project.exe`)
You can add `-e` to skip to the editor, `-g` for gameplay, `-deez` to skip the logo screen straight to the main menu, and `-o` for the options.
