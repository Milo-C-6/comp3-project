## Compiling

1. Clone the code
    1. `git clone https://github.com/Milo-C-6/comp3-project.git`
2. Install w64devkit from [/skeeto/w64devkit/](https://github.com/skeeto/w64devkit/releases/latest) and make sure its on path
3. Setup cmake configuration
    1. `cmake -S . -B build`
4. Build
    1. `cmake --build build`

Executable file will be in  (`../build/${PROJECT_NAME}/${PROJECT_NAME}`)
