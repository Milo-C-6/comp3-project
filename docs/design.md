## Part attributes 

Optional attributes that can be put onto parts to change the world or player

## WIN

Finishes the screen, goes to next level

### KILL

Restarts the screen

### BOUNCY

Bounces the player

### LAUNCHER (int: launch amount)

Launches the player upwards

### Lock block

A block that becomes non-collidable after the key item is picked up

### Pressure plate

Block that changes another block only while a player is above.
